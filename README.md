# Timezone Solr with the data from Natural Earth

https://www.naturalearthdata.com/downloads/10m-cultural-vectors/

Timezones

## QGIS output setting
* Format : CSV
* CRS : WGS84
* Layer Option
  * `CREATE_CSVT`: `NO`
  * `GEOMETRY`: `AS_WKT`
  * `LINEFORMAT`: `LF`
  * `SEPARATOR`: TAB
  * `WRITE_BOM`: `NO`
